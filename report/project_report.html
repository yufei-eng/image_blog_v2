Looki-style 图文Blog & 生活漫画 Skill

项目报告 · 2026年04月13日 · Claude Code + Gemini 3 Pro / 3.1 Flash Image

一、项目概述

本项目搭建了两个 Claude Code Skill，复刻 Looki App 的核心内容生成功能：

photo-blog：基于用户照片批量理解 → 高光素材筛选 → 图文Blog生成（标题/描述/洞察/建议）

life-comic：基于用户照片场景提炼 → 分镜脚本设计 → AI漫画多宫格生成 → 情感叙事文案

理解模型: Gemini 3 Pro 生成模型: Gemini 3.1 Flash Image API: Compass LLM Proxy

二、技术架构与 Workflow

2.1 图文 Blog Workflow

1
收集照片 → 支持目录/文件输入，自动过滤非图片文件
2
批量理解 (Gemini 3 Pro) → 每批5张，自动降采样；输出场景/人物/动作/情感/地点结构化JSON
3
多维评分 + 多样性选取 → 5维评分体系(视觉/叙事/情感/独特/技术) + 贪心多样性优化算法
4
文案生成 (Gemini 3 Pro) → 诗意标题 + 连贯叙事 + 逐张洞察 + 场景化建议
5
HTML渲染 → 深色卡片式布局，内嵌base64图片，独立HTML无外部依赖

2.2 生活漫画 Workflow

1
收集照片 → 同上
2
漫画潜力分析 (Gemini 3 Pro) → 评估每张照片的漫画改编潜力(动态感/视觉区分/叙事分量)
3
分镜素材选取 → 多样性优化选取6个高光时刻，覆盖情感/环境/时间多维度
4
分镜脚本生成 (Gemini 3 Pro) → 主题提炼 + 情感弧线 + 每帧画面描述 + 构图建议
5
漫画图片生成 (Gemini 3.1 Flash Image) → 6宫格多参考图生成，温暖手绘插画风格
6
情感叙事 + HTML渲染 → 散文体正文 + 深色主题布局

2.3 模型/Workflow选型理由

Gemini 3 Pro (理解)：支持单次请求最多14张图片输入，1M token上下文窗口，适合批量照片理解和结构化分析。相比Flash版本，Pro在细粒度场景描述和情感分析上准确度更高。

Gemini 3.1 Flash Image (生成)：支持最多14张参考图输入，可维持5个角色的一致性，准确渲染中文文字，生成速度快。特别适合基于真实照片的风格化漫画生成。

评分 + 多样性选取算法：借鉴 ai-instagram-organizer (GitHub 高星项目) 的 PhotoScore 多维评分体系和 SmartPostCreator 的多样性贪心算法，确保选出的素材既高质量又有足够的场景覆盖。

HTML 独立渲染：参考 gemini_picturebook_generator (42 stars) 的 HTML/PDF 输出模式，图片base64内嵌，单文件即可分享。
