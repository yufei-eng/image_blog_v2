# Image Blog & Life Comic Skills

> Transform photos into narrative-driven visual blogs and warm hand-drawn comic strips.

Powered by **Gemini 3 Pro** (image understanding) + **Gemini 3.1 Flash Image** (comic generation). Works in both **Claude Code** and **Cursor**.

## Quick Install

```bash
bash <(curl -s https://raw.githubusercontent.com/yufei-eng/image_blog/main/install.sh)
```

This will:
1. Clone the repo to `~/.local/share/image_blog`
2. Create symlinks in `~/.claude/skills/` and `~/.cursor/skills/`
3. Install Python dependencies (`google-genai`, `Pillow`)
4. Generate config templates (you fill in your API token)

### Configuration

After install, edit the config file with your Compass API token:

```bash
# Both skills share the same config format
vim ~/.local/share/image_blog/skills/photo-blog/config.json
vim ~/.local/share/image_blog/skills/life-comic/config.json
```

## Two Skills

### photo-blog ‚Äî Photo Blog Generator

Transform photos into evocative visual stories with poetic titles, scene descriptions, photo insights, and personalized tips.

```bash
python3 ~/.claude/skills/photo-blog/main.py ~/photos/ --theme "culinary adventure"
```

### life-comic ‚Äî Life Comic Generator

Transform photos into warm, heartfelt multi-panel comics with emotional prose narrative.

```bash
python3 ~/.claude/skills/life-comic/main.py ~/photos/ --panels 6 --theme "city nightscape"
```

### Natural Language (in Claude Code / Cursor)

Just talk to the agent:
- "Summarize my recent photos as a blog"
- "Turn these photos into a comic strip"
- "Make a travel diary from my vacation photos"
- "Create a food journey comic"

## Key Features (v0.2)

| Feature | Details |
|---------|---------|
| **Flexible image count** | 1-9 photos per generation. Comic grid adapts: 1x1 / 2x2 / 2x3 / 3x3 |
| **Theme keywords** | `--theme "food journey"` guides generation. Falls back with 3 suggested alternatives if photos don't match |
| **Triple output** | HTML (self-contained page) + Rich Text (Markdown for chat) + PNG (shareable composite). All generated by default |
| **HiDPI PNG** | 2x resolution (2160px wide) for crisp display on Retina screens |
| **EXIF handling** | Auto orientation fix + date extraction from EXIF metadata / filename |
| **Diversity algorithm** | Greedy selection balances quality and variety ‚Äî avoids repetitive scenes |
| **Intent recognition** | Agent auto-triggers on "life summary", "daily recap", "memory collage" etc. |
| **Post-gen suggestions** | After generation, suggests comic/blog crossover and theme alternatives |

## Architecture

```
Photo input ‚Üí Gemini 3 Pro understanding ‚Üí Multi-dimensional scoring ‚Üí Diversity selection
                                                    ‚Üì
                                       photo-blog: Narrative ‚Üí HTML + MD + PNG
                                       life-comic: Storyboard ‚Üí Comic image gen ‚Üí HTML + MD + PNG
```

## Update

```bash
bash ~/.local/share/image_blog/update.sh
```

Or re-run the install command ‚Äî it detects existing installations and pulls latest.

## Arguments Reference

### photo-blog

| Arg | Description | Default |
|-----|-------------|---------|
| `input` | Image directory or file path | required |
| `--max-highlights` | Number of highlight photos (1-9) | 9 |
| `--output` | Output file path | `blog_output.html` |
| `--theme` | Theme/style keyword | auto |
| `--format` | `html` / `richtext` / `png` / `all` | `all` |
| `--date` | Footer date (auto from EXIF) | auto |
| `--save-analysis` | Save analysis JSON | none |

### life-comic

| Arg | Description | Default |
|-----|-------------|---------|
| `input` | Image directory or file path | required |
| `--panels` | Number of comic panels (1-9) | 6 |
| `--output` | Output file path | `comic_output.html` |
| `--theme` | Theme/style keyword | auto |
| `--format` | `html` / `richtext` / `png` / `all` | `all` |
| `--output-dir` | Directory for comic images | `.` |
| `--skip-image-gen` | Skip comic image generation | false |
